This example demonstrates an online simulation of MicrosoftTM Visual Studio .Net IDE - base version 7.1.3088. The example shows integration between r.a.d.menu, r.a.d.toolbar and r.a.d.dock To experience the versatility and power of telerik r.a.d.dock, play along with these toolbar buttons:

  • - toggle toolbox visibility
  • - toggle output window visibility
  • - simulate successful / unsuccessful build
  • - new project (F5)
  • - toggle editor word-wrap
  • - close editor

Example highlights:

  • Complete MicrosoftTM Visual Studio .Net resemblance
  • Interactive and intuitive user interface - no 'compilation' could be performed if there is no open project, etc.
  • Functioning toolbar and menu command buttons
  • Prompts and alerts
  • Ability of capturing client events and exchanging data among the different controls - r.a.d.docks, iframes, input fields, id's etc.
  • Crossbrowser compatibility (MSIE, Mozilla, Opera)
  • Editor area is editable (MSIE only) and syntax highlighted
  • Toggle word-wrap editor option
  • Functional statusbar
