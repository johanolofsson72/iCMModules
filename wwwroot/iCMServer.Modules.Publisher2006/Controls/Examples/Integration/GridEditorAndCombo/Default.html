This example defines some custom editors for a r.a.d.grid control.  The editors used are:

  • r.a.d.editor for rich text editing
  • r.a.d.combobox for drop down lists 
  • r.a.d.input for the date picking box

You have to inherit the GridTextColumnEditor class and setup a RadEditor object to do the text editing.  Similar logic is required for the drop down list and the RadComboBox -- the base class is GridDropDownColumnEditor.
