This example demonstrates integration between r.a.d.callback controls suite and r.a.d.dock. You can manage the state and appearance of r.a.d.dock through AJAX-enabled controls.

With r.a.d.callback controls you can update any control on the page either in a dockable object or outside of it. The callback controls can be used from inside a dock template.

In the example you can save multiple docking configurations on the server and later reload them.
