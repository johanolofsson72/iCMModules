This example demonstrates integration between r.a.d.callback controls suite and r.a.d.tabstrip and r.a.d.multipage. You can manage the state and appearance of r.a.d.tabstrip and r.a.d.multipage through AJAX-enabled controls.

You can insert callback controls inside r.a.d.multipage templates and use them to update any content on the page.

r.a.d.tabstrip can also be enabled to make callbacks by using the generic r.a.d.callback control. In the example you can bind the tabstrip to XML file data source and add/remove/disable tabstrip items.
